# PTC Agents Documentation

> **Status:** Planning phase (branch: `spike-subagent`)

## Overview

PTC Agents extends PtcRunner with **SubAgents** - isolated workers that execute missions using PTC-Lisp programs generated by an LLM. This enables:

- **Agentic loops** - Multi-turn LLM interactions with tool calling
- **Typed contracts** - Signatures define inputs/outputs for validation and chaining
- **Context isolation** - Each agent has its own tools, context, and memory
- **Composability** - Agents can be nested as tools within other agents

```elixir
{:ok, step} = SubAgent.run(
  "Find top 3 customers by revenue",
  llm: my_llm,
  signature: "() -> {customers [{name :string, revenue :int}]}",
  tools: %{"get_customers" => &MyApp.get_customers/1}
)
```
